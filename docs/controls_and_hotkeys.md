# Controls and Hotkeys

***

**Controls:** 

- Left mouse to place tiles and select options
- Right mouse to delete tiles
- Right click and drag to move camera

**Hotkeys:**
- **Z** = Undo
- **Y** = Redo
- **+** = Zoom in
- **-** = Zoom out
- **0** = Zoom reset
- **F** = Toggle Fill mode
